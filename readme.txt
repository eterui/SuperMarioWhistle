ReadMe Super Mario Whistle Version 1.0 04/30/2014



Requirements to run

-python version 2.7

-pyAudio v0.2.8 for Python 2.7 which can be downloaded at: 
http://people.csail.mit.edu/hubert/pyaudio/

-NumPy v1.8.1 for Python 2.7 which can be downloaded at:
http://sourceforge.net/projects/numpy/files/NumPy/



Running Super Mario Whistle World

For best results, play in a quiet environment.

Super Mario Whistle may be run using a Python interpreter and
calling Game().run()

Begin by clicking Instructions to read the instructions, 
or press Play Game to begin playing.



Special Controls

Press R at any time to restart.

Press S to skip to the boss fight for testing purposes
